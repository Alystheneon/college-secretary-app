# College Secretary App

Εφαρμογή γραμματείας κολλεγίου, γραμμένη σε Java (Swing) με βάση δεδομένων MySQL.
Διαχειρίζεται τους καθηγητές και τα μαθήματα που διδάσκει ο καθένας.

## Πώς τρέχει

1. Χρειάζεσαι JDK 21, MySQL και Eclipse.
2. Τρέξε το `database.sql` για να φτιαχτεί η βάση `HR_app`.
3. Άνοιξε το `DBConnection.java` και βάλε το δικό σου MySQL username/password.
4. Άνοιξε το project στο Eclipse και τρέξε το `App.java`.

## Τι κάνει

- Αναζήτηση καθηγητή με το επώνυμο
- Εισαγωγή νέου καθηγητή
- Εμφάνιση όλου του προσωπικού με τα μαθήματά τους
- Ανάθεση και αφαίρεση μαθημάτων, με έλεγχο ότι δεν ξεπερνιούνται οι διαθέσιμες ώρες του καθηγητή
- Ενημέρωση και διαγραφή καθηγητή

## Βάση δεδομένων

Η εφαρμογή θέλει μια βάση MySQL με όνομα `HR_app` και δύο πίνακες, `Teachers` και
`Courses`. Όλα φτιάχνονται με το `database.sql`.

Τα στοιχεία σύνδεσης (URL, username, password) είναι μαζεμένα στο αρχείο
`CollegeSecretaryApp/src/secretaryapp/database/DBConnection.java`. Άλλαξέ τα ώστε
να ταιριάζουν με τη δική σου εγκατάσταση MySQL. Αν η σύνδεση αποτύχει, η εφαρμογή
βγάζει μήνυμα και κλείνει.

## Πώς το τρέχεις

Ο πιο εύκολος τρόπος είναι μέσω Eclipse: File > Import > Existing Projects,
διάλεξε τον φάκελο `CollegeSecretaryApp` και μετά Run As > Java Application πάνω
στο `App.java`.

## Δομή του κώδικα

- `App.java` — το σημείο εκκίνησης, ανοίγει την αρχική οθόνη
- `BaseForm.java` — η βασική κλάση που κληρονομούν όλα τα παράθυρα (τίτλος, μήνυμα,
  κουμπί εξόδου), για να μην ξαναγράφεται ο ίδιος κώδικας παντού
- `DBConnection.java` — η σύνδεση με τη βάση, κοινή για όλες τις οθόνες
- `forms/` — οι οθόνες:
  - `Home` — αρχική
  - `NewTeacher` — εισαγωγή καθηγητή
  - `Personnel` — πίνακας με όλο το προσωπικό
  - `Results` — αποτελέσματα αναζήτησης, ενημέρωση και διαγραφή
  - `Courses` — ανάθεση μαθημάτων
  - `Version` — έκδοση εφαρμογής

Οι δύο πίνακες της βάσης έχουν σχέση ένας-προς-πολλά: ένας καθηγητής μπορεί να
διδάσκει πολλά μαθήματα, και κάθε μάθημα έχει το πολύ έναν καθηγητή. Όταν
διαγράφεται καθηγητής, τα μαθήματά του δεν σβήνονται, απλώς μένουν χωρίς καθηγητή.

## Σημειώσεις

- Το εικονίδιο του παραθύρου στο `BaseForm.java` φορτώνεται από μια σταθερή
  διαδρομή των Windows, οπότε σε άλλον υπολογιστή απλά δεν θα εμφανίζεται. Δεν
  επηρεάζει τίποτα άλλο στη λειτουργία.
- Τα παράθυρα έχουν σταθερό μέγεθος και δεν αλλάζουν διαστάσεις.
